本Demo是基于jQuery、eCharts以及less完成的，其完成的效果图如下所示。

需要注意的点是：

1、由于采用了IIFE进行隔离变量，所以每次后面必须加分号，不然会报错(找了很久的坑)

2、less编写，推荐使用VScode的插件easy less进行翻译为css；

3、地图增加城市，搜到你想加的城市经纬度，加入到index.js中geoCoordMap里面

4、时钟：可以采用动画或者定时器进行设置
